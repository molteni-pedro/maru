This plugin is used to insert data table using elementor editor.

== Description ==

Datalentor is an elementor add-on to showcase data with tabular format and beautiful way. This is an simple and flexible way to decorate your data analysis in table style with the help of Elementor page builder.

### Features

We have researched all basic requirements to present anything with **Data Tables**.

You can create and manage tables from the admin panel quickly, effectively and easily.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==
= Can I add multiple Datalentor widgets with single page? =
Yes, you can. Datalentor plugin gives you the ability to add **multiple Data Tables on same page**. You just need to drag and drop that module multiple times and input different data values whatever you want to showcase.

=  Can I showcase button too instead of text input only? =
Yes, You can. We have provided button input value too that will be helpful to showcase button instead of text with data tables.
